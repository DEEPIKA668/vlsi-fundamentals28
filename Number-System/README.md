## Objective

The objective of this module is to understand different number systems used in digital electronics and VLSI design, along with various conversion techniques between them.

# Why Number Systems are Important in VLSI?

Digital circuits understand only two logic levels:

- Logic 0
- Logic 1

Every digital circuit, processor, memory, FPGA, ASIC, and microcontroller internally works using binary numbers. Therefore, understanding number systems is the first step toward learning RTL Design and Verification.

# Number Systems Covered

## 1. Decimal Number System (Base-10)

- Digits: 0 – 9
- Radix: 10
- Used in everyday calculations.

## 2. Binary Number System (Base-2)

- Digits: 0 and 1
- Radix: 2
- Used in all digital electronic systems.

## 3. Octal Number System (Base-8)

Digits:

0 – 7

Used for compact representation of binary numbers.

## 4. Hexadecimal Number System (Base-16)

Digits:

0 – 9

A B C D E F

A = 10

B = 11

C = 12

D = 13

E = 14

F = 15
